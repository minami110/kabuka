# kabuka
* https://minami110.github.io/kabuka/


## for developers
* use docker (docker-compose)

### clone
* `git clone https://github.com/minami110/kabuka/`

### run dev server
* `cd kabuka`
* `docker-compose up -d`
* goto `http://localhost:3000/kabuka/`

### deploy to github pages
* push or merge into `origin/pre_deploy` branch
* ci: TravisCI
* coverage: Coverall
